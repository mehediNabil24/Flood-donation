Important FAQ

What is the Document Object Model(DOM)?

The DOM (Document Object Model) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, where each element (like headings, paragraphs, and buttons) is a node. This allows programming languages like JavaScript to dynamically access, manipulate, and update the content, structure, and styles of a webpage

How do you select an element from the DOM?

By document.getElementById('id') or documnet.querySelector('selector')

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where instead of attaching event listeners to individual child elements, you attach a single event listener to a parent element. This listener can then handle events for all child elements, taking advantage of event bubbling

How do you manipulate an element's attribute and styles using the DOM?

To manipulate an element's attributes in the DOM:
Use getAttribute() to retrieve an attribute's value.
Use setAttribute() to set or change an attribute's value.
Use removeAttribute() to remove an attribute from the element.
To manipulate an element's styles:
Use the style property to change inline styles (e.g., element.style.color = 'blue';).
Use style.cssText to set multiple styles at once.
